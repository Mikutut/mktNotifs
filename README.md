# mktNotifs

## Mikut's notifications system

### Installation

### When using with FiveM 
1. Download latest release from `"Releases"` tab
2. Unpack contents of `"FiveM"` anywhere in `/resources` folder
3. Add `ensure mktNotifs` in your `server.cfg` file.
4. Enjoy!
